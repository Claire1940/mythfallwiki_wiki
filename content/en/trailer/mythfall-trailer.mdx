To help you prepare for the mechanical depth shown in the trailer, familiarize yourself with the default control layout:

| Control Type | PC Input | Controller Input | Action & Combat Function |
| :--- | :--- | :--- | :--- |
| **Sprinting** | **Left Shift** | **L3 (Left Stick Click)** | Increases movement speed; essential for dodging wide AoE indicators. |
| **Dodging** | **Q Key** | **B Button / Circle** | Provides brief invincibility frames; mandatory for avoiding unblockable attacks. |
| **Defending** | **F Key** | **L2 Button / LT** | Blocks standard attacks. Timed perfectly, this triggers a parry to stun foes. |
| **Skills** | **Z / X / C / V** | **D-Pad / Face Buttons** | Triggers equipped weapon skills, magic spells, and character abilities. |

<Callout type="tip" title="Mastering the Parry Mechanic">
Parrying is the single most important defensive tool in high-level PvE. Successfully parrying an opponent negates all damage and inflicts a brief posture break, allowing you to chain heavy critical strikes.
</Callout>
