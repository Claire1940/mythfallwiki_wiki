Para ajudar você a se preparar para a profundidade mecânica mostrada no trailer, familiarize-se com o layout de controles padrão:

| Tipo de Controle | Entrada PC | Entrada de Controle | Função de Ação e Combate |
| :--- | :--- | :--- | :--- |
| **Correr** | **Left Shift** | **L3 (Clique no Analógico Esquerdo)** | Aumenta a velocidade de movimento; essencial para desviar de indicadores de área (AoE). |
| **Esquivar** | **Tecla Q** | **Botão B / Círculo** | Fornece breves frames de invencibilidade; obrigatório para evitar ataques imbloqueáveis. |
| **Defender** | **Tecla F** | **Botão L2 / LT** | Bloqueia ataques padrão. Sincronizado perfeitamente, ativa uma aparagem para atordoar inimigos. |
| **Habilidades** | **Z / X / C / V** | **D-Pad / Botões de Face** | Ativa habilidades de armas equipadas, feitiços mágicos e habilidades de personagem. |

<Callout type="tip" title="Dominando a Mecânica de Aparagem">
A aparagem é a ferramenta defensiva mais importante no PvE de alto nível. Aparar um oponente com sucesso anula todo o dano e inflige uma breve quebra de postura, permitindo que você encadeie ataques críticos pesados.
